# ZeldaGroupe – Projet Java 2 (Travail collaboratif)
## Objectif du projet
Ce dépôt correspond à un projet réalisé dans un cadre **collaboratif étudiant**, en **Programmation Orientée Objet**
### Le projet met en œuvre les notions fondamentales de Java 2 :
* **Classes abstraites** : définition de comportements communs (GameObject, Character)
* **Polymorphisme** : comportements spécialisés selon le type d’objet
* **Encapsulation** : accès contrôlé aux attributs et méthodes
* **Logique de collision algorithmique** : implémentation progressive sans dépendance externe
### Le projet met en œuvre les notions fondamentales de Java 2 :
* **Héritage** : organisation hiérarchique des entités du jeu

* Intégration progressive des **remarques** pédagogiques du **professeur**
